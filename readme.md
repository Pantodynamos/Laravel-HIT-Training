HIT Laravel Training Project

-FIFO Stocks App

-XAMPP v3.2.4

-Laravel v5.8

How to use :

Use XAMPP or your preferred choice
Run Apache + MySQL if using XAMPP
Migrate database ( php migrate )
Insert data for product, location, and program table
Use command php artisan serve to start local server
Use your web browser to access this url http://127.0.0.1:8000
Use the web UI to interact with the program
